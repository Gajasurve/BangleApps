To create a proper icon:

1. Create a 48x48 PNG image with:
   - Om symbol (‡•ê) in center
   - Saffron/orange color (#FF9933)
   - Or a simple mandala pattern
   - Or a lotus flower

2. Go to: http://www.espruino.com/Image+Converter

3. Upload your PNG

4. Select:
   - Image type: "1 bit Black and White"
   - Compression: "Heatshrink (RLE)"
   - Output as: "Image String"

5. Copy the generated code

6. Save as panchang.img
