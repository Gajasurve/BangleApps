# Vedic Panchang Clock

Hindu Panchang watchface for Bangle.js 2.

## Features

- Large saffron time display
- Current date
- Hindu masa (month) name
- Tithi with paksha and nature (K9(Rkt) format)
- Colorful Ekadashi countdown box
- Daily Vishnu Sahasranama name
- Battery % and steps
- Hourly vibration alerts

## Display

- **Top**: Battery | Steps
- **Masa**: Hindu month (Margashirsha, Pausha, etc.)
- **Time**: HH:MM (large, saffron)
- **Date**: "13th Dec"
- **Tithi**: S5(Prn) = Shukla 5, Purna | K9(Rkt) = Krishna 9, Rikta
- **Ekadashi**: Days countdown in colorful box
- **Vishnu**: Daily name from Sahasranama

## Tithi Nature

- Nan (Nanda): Joyful
- Bhd (Bhadra): Auspicious
- Jay (Jaya): Victorious
- Rkt (Rikta): Empty
- Prn (Purna): Complete

## Battery

- Screen off after 5 seconds
- Updates every 60 seconds
- Wake on wrist twist
- 3-5 days battery life
